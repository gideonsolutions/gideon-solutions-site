# Gideon Solutions

Professional services for individuals and businesses — [gideonsolutions.us](https://gideonsolutions.us)

## Services

- **Tax Returns** — Individual income tax preparation and filing (federal and state)
- **Websites** — Professional website creation and ongoing maintenance
- **IT Support** — Corporate machine preparation and configuration (Windows, Mac, Linux)
- **Custom Work** — Software engineering, data engineering, and data science solutions

## Tech Stack

- [Next.js](https://nextjs.org) (App Router)
- [Tailwind CSS](https://tailwindcss.com)
- [Resend](https://resend.com) (contact form emails)
- Deployed on [Vercel](https://vercel.com)

## Development

```bash
npm install
npm run dev
```

Requires a `.env` file — see [`.env.example`](.env.example) for required variables.
